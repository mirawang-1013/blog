## Why now?

I've been taking notes in Obsidian for months — paper readings, interview prep, LeetCode solutions, career reflections. But notes that stay private don't compound. Writing in public forces clarity.

## What is this blog?

This is a Next.js static blog powered by markdown files. But the interesting part is **how** the content gets here:

I built an **Obsidian Agent** — a Python tool that:
1. Reads my Obsidian vault
2. Uses RAG to find relevant notes
3. Generates blog-ready drafts via an LLM

The agent doesn't replace my writing. It gives me a first draft to react to, which is infinitely easier than staring at a blank page.

## What I'll write about

- **AI/ML**: Paper readings (CNN series, Transformers), RL experiments
- **Building**: Open source projects, tools, side projects
- **Career**: Job search reflections, interview prep notes
- **Learning**: Things I figure out that might help someone else

## The meta part

This entire blog — the Next.js setup, the Obsidian Agent, and this first post — was built in a single afternoon. I used anxiety as fuel and flow as the vehicle.

Sometimes the best cure for overthinking is to ship something.
